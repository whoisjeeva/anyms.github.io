How to Enable Chrome Extension to Run on Incognito

Jeeva
Apr 28, 2024 • Google Chrome

Google Chrome extensions are programs that can be installed into Chrome in order to change the browser's functionality. This includes adding new features to Chrome or modifying the existing behavior of the program itself to make it more convenient for the user.

Step 1: Finding a Chrome Extension to Enable in Incognito Mode

You might have a lot of extensions installed on your Google Chrome browser, if you want to allow an extension to run in incognito mode. First you have locate the extension. In order to do that, navigate to chrome://extensions, on this page you can see all of your installed Google Chrome extensions.

Step 2: Enable a Google Chrome Extension to Run in Incognito Mode

After you located the extension you want to enable to run in incognito mode, click on the details button. There you will see all the details & permissions of that extension.

There will be a toggle option called Allow in Incognito if you don't see one, scroll down. By toggling Allow in Incognito you can tell the Google Chrome browser to enable the extension in incognito mode or not.

WARNING: Google Chrome cannot prevent extensions from recording your browsing history. To disable this extension in Incognito mode, unselect this option.
